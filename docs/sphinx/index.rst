******
Umpire
******

Umpire is a resource management library that allows the discovery, provision,
and management of memory on next-generation hardware architectures with NUMA
memory hierarchies.

- Take a look at our Getting Started guide for all you need to get up and
  running with Umpire.

- If you are looking for developer documentation on a particular function,
  check out the code documentation.

- Want to contribute? Take a look at our developer and contribution guides.

Any questions? File an issue on GitHub, or email umpire-dev@llnl.gov
